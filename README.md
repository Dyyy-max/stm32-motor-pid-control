# STM32F103 直流电机PWM调速 + PID闭环稳速

## 项目简介

本人是一名大三电子信息工程专业在校生。

本项目是在学校自主完成的嵌入式综合实践练习。

基于STM32F103C8T6(HAL库)开发的嵌入式车载电机控制Demo，
实现按键启停、LED状态指示、PWM调速、增量式PID转速闭环稳速功能，
所有运行数据可通过串口实时打印、观测调试。

项目对标汽车电子电机控制场景。
用于巩固嵌入式外设开发、算法调试能力，为嵌入式实习积累工程经验。

## 技术栈

- 主控芯片：STM32F103C8T6
- 开发工具：STM32CubeMX、Keil MDK5
- 外设：GPIO、TIM(PWM)、USART串口
- 核心算法：增量式PID闭环控制
- 版本管理：Git / GitHub

## 文件说明

| 文件 | 说明 |
|------|------|
| `main.c` | 硬件版主程序，基于HAL库外设开发，支持真实电机硬件驱动 |

## 硬件说明

本项目无需实体硬件，全程基于 Keil MDK 软件仿真运行。

- 仿真目标芯片：STM32F103C8T6
- 可仿真外设：GPIO、TIM(PWM)、USART 串口

## 功能列表

1. 按键控制电机启停，LED同步状态提示
2. TIM2 输出1kHz PWM波形调节电机转速
3. 增量式PID算法实现转速闭环稳速
4. USART1(9600波特率)串口实时打印转速、PWM占空比、PID参数

## 编译&运行步骤

### 硬件版 (main.c)

1. 使用 STM32CubeMX 配置时钟、定时器、串口、GPIO
2. 将 `main.c`、`pid.c`、`pid.h` 添加到 Keil 工程
3. 编译后下载到 STM32 开发板
4. 串口助手设置 9600 波特率，查看运行数据

### 仿真版 (main_simulation.c)

1. Keil MDK5 新建工程，目标芯片选择 STM32F103C8
2. 添加 `main_simulation.c`、`pid.c`、`pid.h`、`startup_sim.s`
3. 开启软件仿真模式运行程序
4. 在串口窗口观察 PID 响应曲线数据

## 开发问题与优化解决

### 1. 电机转速震荡、启动超调严重

调试初期 PID 参数不合理，导致转速波动大、启动冲值过高。

**解决方案：**
- 多次整定 Kp、Ki、Kd 参数
- 增加积分限幅、输出限幅机制
- 最终系统转速运行平稳、响应稳定

### 2. 硬件依赖高、调试效率低

最初必须依赖硬件才能验证算法，调试成本高。

**解决方案：**
- 新增纯软件仿真版本，脱离硬件环境
- 可快速验证 PID 逻辑、对比参数效果
- 大幅提升开发效率

### 3. 代码结构杂乱、可读性差

初版代码逻辑堆砌在主循环中。


**解决方案：**
- 优化代码结构，拆分业务逻辑
- 规范注释，符合嵌入式开发规范
